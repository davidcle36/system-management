USchedule, allow the users to add, update, and delete customers along with their respective appointments to be made.
David Le, dle36@wgu.edu, v1.0.1, January 6, 2021
Intellij 2020.3.1, java 11.0.9 2020-10-20 LTS, JavaFX-SDK-11.0.2
Run the program in intellij by pressing run button.
    Record button in the main view, after logging in, which generate report.
    Can switch tab between customers and appointments. The customers and appointments shows all by items by default.
Additional report included information for each countries, it shows the customer information
    such as address, name, id postal code, and their division.
